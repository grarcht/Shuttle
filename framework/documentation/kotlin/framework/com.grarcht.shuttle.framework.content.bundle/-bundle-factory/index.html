interface BundleFactory

This contractual interface is used to provide a factory that Bundle objects. Having this factory enables for unit testing of Bundle objects in the Shuttle Framework.

For more information on the factory design pattern, refer to: <a href="https://www.tutorialspoint.com/design_pattern/factory_pattern.htm">Factory Design Pattern</a>
