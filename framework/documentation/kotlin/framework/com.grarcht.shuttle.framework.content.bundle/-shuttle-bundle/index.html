open class ShuttleBundle(repository: ShuttleWarehouse, internalBundle: Bundle?)

This class creates stores a Bundle cargo in the ShuttleWarehouse and creates a Bundle that can be transported with avoiding the Transaction Too Large Exception. This class uses the Fluent Interface Design Pattern to achieve function chaining. For more information on this design pattern, refer to: <a href="https://en.wikipedia.org/wiki/Fluent_interface">Fluent Interface</a>
