ShuttleIntent

open class ShuttleIntent

This class creates stores the intent's data cargo in the ShuttleWarehouse and creates an Intent that can be transported with avoiding the Transaction Too Large Exception. This class uses the Fluent Interface Design Pattern to achieve function chaining. For more information on this design pattern, refer to: Fluent Interface
