getServiceName
Link copied to clipboard
open fun getServiceName(): String

Override this function to provide a human-readable string. For obfuscated apps, this string will become garbled since reflection is used to get the name of the class. When looking at log tags in log management systems, the tag will not be readable. The recommended approach is to use the name of your service in a string. For instance, if your service is named MyService, then return "MyService".
