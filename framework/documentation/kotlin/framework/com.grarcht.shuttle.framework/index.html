interface Shuttle
Content copied to clipboard

This contractual interface is used to provide a simple way to shuttle parcelable data and avoid app crashes from Transaction Too Large Exceptions. This interface is a factory that creates ShuttleBundle and ShuttleIntent cargo objects. It is these objects which are created and access the repository to persist the data and return a small object to use in Android parcel transactions. Additionally, this interface is a producer used to produce persisted Bundles. The intent of this interface is to enforce implementations to provide the consumer portion of the Producer - Consumer Design Pattern.
