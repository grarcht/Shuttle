The contractual interface for adapting Parcelables to ByteArrays and vice-versa. This class follows the Adapter Design Pattern to adapt objects for interaction with the Shuttle databases. For more information on this design pattern, refer to: <a href="https://www.tutorialspoint.com/design_pattern/adapter_pattern.htm">Adapter Design Pattern</a>
