open class ShuttleRoomDao(dao: ShuttleRoomDao.Dao) : ShuttleDataAccessObject

This data access object class takes inspiration from the Decorator and Template Design Patterns to decorate the Room Dao object with a constant to represent the insertion fail state and to ensure the behavior the Room Dao object performs the what is expected for ShuttleDataAccessObjects. For more information on these design patterns, refer to: Decorator Design Pattern Template Design Pattern Data Access Object (DAO) Design Pattern
